### 关键特性

* 480 MHz C906 + 320MHz E907 + 160MHz E902
* 100GOPS NPU(BLAI-100)
* MJPEG and H264(Baseline/Main)
* Wi-Fi 802.11 b/g/n Wi-Fi4
* Bluetooth 5.x Dual-mode BT + BLE
* USB HS OTG
